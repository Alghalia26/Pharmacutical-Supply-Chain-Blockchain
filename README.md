# Pharmacutical-Supply-Chain-Blockchain
Implementation of a blockchain-based pharmaceutical supply chain framework using Hyperledger Fabric, integrating traceability, counterfeit medicine verification, IoT-enabled temperature monitoring, inventory control, and performance benchmarking with Hyperledger Caliper. This repository is divided into three main parts:
hyperledger-fabric: contains the blockchain network configuration and smart contract implementation.
iot-temperature-sensor: contains the simulated IoT temperature monitoring script.
hyperledger-caliper: contains the benchmarking configuration, workload files, and performance results.
